# 辅导员值班系统

​        前端部分有两份 一份运行在微信小程序界面（手机端）可以实现辅导员级学院级管理员级三级角色对各个辅导员的值班换班管理 一份运行在pc端 仅支持校级管理值班系统且双端链接同一数据库可以同时查询历史数据 后端代码用java编写



## 小程序端完整需求文档

### 一、项目技术栈

- **前端框架**：微信小程序原生框架

### 二、角色与权限

| 角色   | 权限范围                                     |
| :----- | :------------------------------------------- |
| 辅导员 | 查看个人值班表、发起换班申请、确认/拒绝换班  |
| 学院   | 本学院所有值班表、审批本学院内换班申请       |
| 管理员 | 全校值班表、手动调整任何值班、审批跨学院换班 |
